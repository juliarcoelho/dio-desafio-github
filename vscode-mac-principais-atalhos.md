# Atalhos VSCode para Macbook


## Abrir/Fechar - Ver
| Funcionalidade   |  Mac | 
|------------------|:------:|
| Todos os atalhos  |  ⌘K ⌘S   |
| Lista de Comandos | ⇧⌘P     |
| Terminal Integrado|  ⌃\`     |
| Sidebar           | ⌘B      |
| Ir para entidade (Func, vars, etc)  |  ⇧⌘O      |
| Ir para linha  |  ⌘G   |



## Editor
| Funcionalidade   |  Mac | 
|------------------|:------:|
| Cortar a linha | ⌘X | 
| Copiar a linha |	 ⌘C | 
| Mover a linha  | ⌥↓ / ⌥↑ |
| Mover cursor por linha  |	 ⌥← / ⌥→ | 
| (Des)Comentar a linha | ⌘/ |	
| Formatar a seleção | ⌘K ⌘F|
| Formatar o documento |⇧⌥F|
| Pesquisar     |  ⌘F |


## Arquivos
| Funcionalidade   |  Mac | 
|------------------|:-----:|
| Novo Arquivo  | ⌘N | 
| Abrir arquivo |  ⌘O | 
| Ir para Arquivo |  ⌘P |
